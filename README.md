### How to Run

1. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

2. Start the application:

   ```bash
   python app.py
   ```

3. Open your browser to: `http://localhost:8000`

- **Voice-First**: Integrated microphone controls and visualizer.
